Hoi, ik ben Daan Snoeken, een 26-jarige Java-programmeur.
Hopelijk kunt u erg genieten van mijn interactieve CV! Op deze pagina zal ik mijn best doen om mezelf voor te stellen.

In maart 2020 ben ik afgestudeerd van de Radboud Universiteit te Nijmegen, met een MSc. in Physical Chemistry. Tijdens deze master heb ik veel ervaring opgedaan met programmeren in onder andere C++, Fortran, Matlab, Bash en ook Java. Dit komt ook vooral door mijn stage bij de afdeling Theoretische Chemie. Hier ben ik ook echt mijn programmeertalent tegengekomen!
Om deze reden heb ik besloten om het Java traineeship bij Young Capital te doen, zodat ik me nog verder hierin door kan ontwikkelen en mijn baan ervan kan maken.

Van binnen ben ik echt een programmeur. Ik wil ook graag mijn vaardigheden in dit werkveld kunnen toepassen.

Het liefste dan als Java developer of als full-stack developer. Ik heb ook aanleg voor een diverse industrie. Mijn intresses lopen dus vrij breed uiteen, van fundamentele wetenschap tot financiën. Ik kan dan ook vrijwel overal mijn weg vinden.

Op mijn GitHub account kunt u een aantal projecten vinden die ik tijdens mijn programmeercarrière uitgevoerd heb. Onder het kopje "Programmeren" wil ik hier dieper op in gaan.

Verder heb ik tijdens het traineeship mijn Scrum.org PSM1 certificaat gehaald met een score van 88%. Vervolgens heb ik ook het Java OCA examen gehaald met een score van 78%.

Ter afsluiting van deze fase van het traineeship heb ik, samen met een team, een full-stack applicatie gebouwd. Hier hebben we een applicatie voor de sport genaamd powerliften gemaakt, waar je als sporter je voortgang kan bijhouden en planningen kan maken.
